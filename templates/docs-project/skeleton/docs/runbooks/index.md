# Runbooks

Operational runbooks for ${{ values.name }}.

## What is a Runbook?

A runbook is a set of standardized procedures for handling common operational tasks or incidents.

## Runbook Template

```markdown
# Runbook: [Title]

## Overview
Brief description of what this runbook covers.

## Prerequisites
- Required access/permissions
- Required tools

## Procedure
1. Step one
2. Step two
3. Step three

## Verification
How to verify the procedure was successful.

## Rollback
Steps to rollback if something goes wrong.

## Contacts
- Primary: @team-lead
- Escalation: @on-call
```
